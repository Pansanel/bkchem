3.2. Export formats

In time of writing of this paragraph BKChem supports export only to SVG (see the section above for more information on BKChem and SVG). There are other exports available which are only experimental and incomplete. These are PNG (cannot export text), PostScript (problems with right postscript font selection) and PovRay (just a funny export not intended for something serious (the rendered scenes can however be very nice)) - see http://www.povray.org for more information about PovRay.
